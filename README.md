# ProjectTracker - Full-Stack Project Management System

A scalable project management application with predictive analytics features.

## Features

- Project creation and lifecycle management
- Task assignment, tracking, and status updates
- Predictive forecasting for project timelines and potential delays
- Role-based access control (Admin, Manager, Team Member)
- Dashboard with key metrics and insights

## Tech Stack

- **Backend**: NestJS (TypeScript) + Modular architecture
- **Database**: MongoDB
- **Frontend**: React.js
- **Authentication**: JWT
- **Others**: RESTful APIs

## Key Highlight

Includes a forecasting module that predicts task duration and project completion risks — developed based on my research work published at IEEE ICCCNT 2025.

## How to Run

### Backend

```bash
cd backend
npm install
npm run start:dev
```
